# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-08-05

### Added
- 초기 릴리스
- HTML5 비디오 플레이어 코어 기능
- TypeScript 지원
- 완전한 재생 컨트롤 (재생/일시정지, 진행바, 볼륨, 전체화면)
- 키보드 단축키 지원
- 다양한 테마 (기본, 다크, 라이트)
- 반응형 디자인
- 이벤트 시스템
- CDN 및 NPM 배포 지원
- 접근성 기능
- 고대비 모드 지원
- 애니메이션 감소 모드 지원

### Features
- 재생 속도 조절
- 볼륨 컨트롤
- 전체화면 모드
- 진행바 드래그 앤 드롭
- 자동 숨김 컨트롤
- 다중 비디오 소스 지원
- 포스터 이미지 지원
- 미리 로드 옵션
- 반복 재생
- 자동 재생

### Browser Support
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
